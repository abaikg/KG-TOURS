# Brand Design System - Nature-Inspired Premium

## 1. Color Palette

### Primary Colors (Nature-Inspired)
**Sage Green (Background & Accents)**
- `sage-50`: #F5F7F5 - Lightest background
- `sage-100`: #E8EDE8 - Card backgrounds
- `sage-200`: #D4E4D4 - Soft accents
- `sage-300`: #C8D5C8 - Main background
- `sage-400`: #A8BFA8 - Borders
- `sage-500`: #7FA87F - Subtle elements

**Forest Green (Text & Dark Elements)**
- `forest-700`: #2D5F4D - Dark green text
- `forest-800`: #1A3A2E - Darkest green
- `forest-900`: #0F2419 - Almost black

**Neutral Warm Tones**
- `cream`: #FDFCFA - Card surface
- `warm-white`: #F5F3EE - Alternative surface
- `charcoal`: #2C3E3B - Primary text

**Accent Colors**
- `gold`: #F4C430 - Stars, highlights
- `amber`: #FFB84D - Warm accents

## 2. Typography

**Font Family:** 'Inter', system-ui, -apple-system, sans-serif

**Scale:**
- Hero: 48px / 1.1 / 800
- H1: 36px / 1.2 / 700
- H2: 28px / 1.3 / 700
- H3: 20px / 1.4 / 600
- Body: 16px / 1.6 / 400
- Small: 14px / 1.5 / 400
- Tiny: 12px / 1.4 / 500

## 3. Spacing

- XS: 8px
- SM: 12px
- MD: 16px
- LG: 24px
- XL: 32px
- 2XL: 48px
- 3XL: 64px

## 4. Border Radius

- SM: 12px - Small elements
- MD: 16px - Inputs, small cards
- LG: 24px - Cards
- XL: 32px - Large cards
- Pill: 9999px - Pills, chips, buttons

## 5. Shadows

- Soft: 0 4px 20px rgba(0, 0, 0, 0.06)
- Card: 0 8px 32px rgba(0, 0, 0, 0.08)
- Float: 0 12px 48px rgba(0, 0, 0, 0.12)

## 6. UI Components Style

**Buttons:**
- Pill-shaped (full rounded)
- Primary: forest-800 background, white text
- Secondary: white background, forest-800 border
- Padding: 16px 32px (large), 12px 24px (medium)

**Cards:**
- Large rounded corners (24-32px)
- Soft shadows
- Cream/warm-white backgrounds
- Image-first design with overlays

**Inputs:**
- Rounded (16px)
- Soft borders (sage-400)
- White background
- Focus: forest-700 border

**Pills/Chips:**
- Full rounded (pill)
- White background with subtle border
- Active state: forest-800 background
- Small padding: 6px 16px

## 7. Design Principles

- **Soft & Organic:** No sharp corners, gentle curves
- **Nature-Inspired:** Earthy tones, natural imagery
- **Premium Feel:** Quality shadows, generous spacing
- **Image-First:** Large, beautiful photography
- **Mobile-Optimized:** Touch-friendly, bottom navigation
